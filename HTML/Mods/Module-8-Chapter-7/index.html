Properties for formatting unordered List

Line-style-type:
The type of bullet thats used for the items in the list.
See the table below for possible values. The default is a disc.
List-style-image:
The URL for an image thats used as the bullet

Values for the list-style-type property of an unordered list

disc solid circle
circle hollow circle
square solid square
none no bullet
